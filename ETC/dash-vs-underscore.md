# Dash냐 or Underscore냐?

## Dash (`-`)
- End user is more familiar with dashes
- Therefore, use the dash for things like urls(`/abount-us`) [구글 권고](https://support.google.com/webmasters/answer/76329?hl=ko)
- Use dash for css classes

## Underscore (`_`)
- Usually for spacing
